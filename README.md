# codex_test

This repository contains a simple example project with a basic Tetris-like game written in Python using `pygame`.

## Requirements

- Python 3
- Install dependencies with `pip install -r requirements.txt`

## Usage

Run the game using:

```bash
python3 tetris.py
```

Use the arrow keys to move and rotate the falling pieces. The down arrow speeds up
the fall. Press `Esc` to quit.
